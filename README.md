SemanticUI Mystique
=============
Copies relevant CSS button styles into [SemanticUI](http://semantic-ui.com/) variable format. Outputs element styles that differ from baseline.


Install
--------
*Not* available in Chrome Extensions store. Clone this repo locally. Follow instructions to [install an unpackaged Chrome
extension](https://developer.chrome.com/extensions/getstarted#unpacked).


How to use
----------
After installing, click the "SemanticUI" icon. Hover over element whose style you want to copy. Ensure dashed red outline is present. Right-click, select "SemanticUI Mystique" option. View output in Chrome Developer Javascript Console.



Known Issues
--------------------
Due to the nature of using the mouse to select an element, it will
always have the `:hover` pseudo-class style. Additionally, in order to
communicate which element is selected, it will always have a red-dashed
`outline` attribute.
